# Bike sharing rental prediction
This project predicts bike sharing rental demand using Machine learning techniques.
## Objective
To analyzehistorical bike rental data and predict future bike demand based on weather conditions, season, temperature, and holidays.
## Tools used
- Python
- Jupyter Notebook
- Pandas
- Numpy
- Scikit-Learn
- Matplotlib
- Seaborn
## Dataset
Bike Sharing Dataset (Kaggle)
## Output
The system predicts number of bikes required for a given day
